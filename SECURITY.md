## Reporting a Vulnerability
I am committed to the security of my labs. If you discover a potential security vulnerability, please do not use the public GitHub issue tracker. 

Instead, please report it privately by:
1. Opening a **Draft Security Advisory** in the "Security" tab of this repository.
2. Alternatively, you can contact me via the email listed on my GitHub profile.

I will acknowledge your report within 48 hours and provide a detailed response with next steps.
